#  Temperature Converter Website

A responsive and interactive Temperature Converter Website developed using HTML5, CSS3, and JavaScript as part of the Oasis Infobyte Web Development and Designing Internship (Task 3).

 Project Objective

Build an interactive web application that converts temperature values between Celsius (°C), Fahrenheit (°F), and Kelvin (K) with proper input validation and accurate conversion formulas.

🚀 Features

- 🌡️ Convert temperatures between Celsius, Fahrenheit, and Kelvin
- 🔢 Numeric input field with validation
- ❌ Displays error messages for invalid or empty input
- 🧊 Prevents values below absolute zero
- 📊 Displays all converted temperature values simultaneously
- 🎨 Clean, responsive, and user-friendly interface
- ⚡ Fast and accurate temperature conversions using JavaScript

 Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Open **index.html** in your web browser.
4. Enter a temperature value, select the input unit, and click Convert.
